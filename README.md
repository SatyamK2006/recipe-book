# CulinaShare - Premium Recipe Book SPA

A modern, production-ready Recipe Book Single Page Application built with React, Vite, Tailwind CSS, and Framer Motion.

## ðŸš€ Features

- **Premium UI/UX**: Glassmorphism effects, smooth page transitions, and micro-interactions.
- **Rich Recipe Data**: 20+ mock recipes with detailed nutrition, ingredients, and steps.
- **Advanced Search & Filtering**: Real-time search with debouncing and category filtering.
- **Recipe Details**:
  - Interactive checkable cooking steps.
  - Collapsible sections.
  - Nutrition facts visualization.
  - Related recipes.
- **Favorites System**: Persisted to LocalStorage.
- **Responsive Design**: Mobile-first approach, looks great on all devices.
- **Performance**: Optimized with code splitting and lazy loading.

- **Framework**: React 18 + Vite

## ðŸ“¦ Installation

1.  **Clone the repository**:

    ```bash
    git clone https://github.com/yourusername/recipe-book.git
    cd recipe-book
    ```

2.  **Install dependencies**:

    ```bash
    npm install
    ```

3.  **Start the development server**:

    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```

## ðŸŽ¨ Customization

- **Colors**: Edit `tailwind.config.js` to change the `primary` and `secondary` color palettes.
- **Fonts**: The project uses `Outfit` and `Playfair Display`. You can change these in `src/index.css` and `tailwind.config.js`.
- **Data**: Add or modify recipes in `src/data/recipes.js`.
